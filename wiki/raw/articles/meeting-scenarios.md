## 通用場景

|                         |                     |                                                                |
|:------------------------|:--------------------|:---------------------------------------------------------------|
| 通用場域                | 類別                | 流程                                                           |
| mybro作為入口串接ITKM   | 文件查詢（RAG應用） | 使用者需手動至ITKM搜尋文件，流程中斷且效率低                   |
| Email附件自動整理       | 附件管理            | 使用者需手動下載附件並分類保存，流程繁瑣                       |
| 會議後任務整理          | 會議管理            | 會議結束後需人工整理待辦事項與分工                             |
| 桌面/資料夾自動歸檔整理 | 檔案管理            | 使用者桌面或下載資料夾檔案雜亂，需手動分類與整理，長期難以維護 |
| 找檔案（本機語意搜尋）  | 檔案查找            | 使用者需依記憶手動搜尋本機檔案，效率低且容易找不到             |
| 下載檔案自動分類        | 檔案管理            | 使用者下載檔案後需手動分類與整理                               |
